# swe266p_bank

## Setup



### Install flask
```
pip install flask
```

### Set environment
Use command line in the directory "swe266p_bank"
- For Mac
```
export FLASK_APP=app
export FLASK_ENV=development
```

- For Windows
```
setx FLASK_APP app
setx FLASK_ENV development
```

## Run
```
flask run
```
or 
```
python -m flask run
```

## Open the website
Open the web browser, go to `http://127.0.0.1:5000`
