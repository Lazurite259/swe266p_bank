# swe266p_bank

# Setup for teammates

```sh

# install flask
pip install flask

# set environment, use cmd in the directory "swe266p_bank"
. For Mac
export FLASK_APP=app
export FLASK_ENV=development
. For Windows
setx FLASK_APP app
setx FLASK_ENV development

# run flask
flask run
or 
python -m flask run
```

# open the website
Open the web browser, go to http://127.0.0.1:5000/
